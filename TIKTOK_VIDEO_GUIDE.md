### Method 3: Desktop Editing (Most Professional)

Use these free tools on Mac:

**iMovie (Free on Mac):**
1. Open iMovie
2. Create new project (9:16 aspect ratio)
3. Import 3 frames
4. Add each for 2-3 seconds
5. Add transitions: Cross Dissolve or Fade
6. Add music from the audio library
7. Export as 1080x1920

**DaVinci Resolve (Free - Pro Level):**
1. Download DaVinci Resolve (free)
2. Set project to 1080x1920
3. Import frames to timeline
4. Add transitions and effects
5. Color grade for consistency
6. Add royalty-free music
7. Export with H.264 codec
